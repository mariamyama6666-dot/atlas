
**Atlas AI Documentary System**

Atlas is an AI-powered system designed to transform an uploaded document into an educational documentary film. The system analyzes the information in the file, explains the key ideas, and automatically searches for suitable images and voice narration to create a coherent documentary that presents the content in a clear and engaging way.

The main goal of Atlas is to convert static written material into a dynamic visual learning experience. By combining narration, visuals, and structured explanation, the system helps transform a silent document into a more interactive and attractive form of learning that supports visual-based understanding and improves knowledge retention.

After generating the documentary, Atlas becomes an interactive AI professor that can discuss the film, answer questions, and guide the user through deeper exploration of the topic. This allows learners to engage with the material more actively and develop a deeper understanding of the educational content.
